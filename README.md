# han-lab-branching-dashboard
R Shiny web application developed for Han Lab (Cornell) dendritic branching quantification.
Key features:

Loads processed lifetime data (BY position & Height/lifetime) from CSV
Compares WT vs PTP69D KO genotypes
Interactive violin + jitter plots for lifetime distribution
Basic spatial soma-to-tip visualization (when position data is available)
Automated outlier detection (Z-score method)

Built to support protocols for branching frequency, lifetime, and second-order branch analysis (kymograph tracing, SNT, ImageJ/FIJI pipelines).
